Aplicação para **converter moedas em tempo real**, consumindo dados de uma API de câmbio.

Projeto desenvolvido para praticar **Vue.js**, com foco em componentização e consumo de API.

## Tecnologias

- Vue.js
- Vite
- JavaScript 
- HTML5 & CSS3
